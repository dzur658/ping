## Updating an iPhone 6  

### 1. Prepare the device  
1. **Charge** – plug the phone into power (or make sure the battery is above 50 %).  
2. **Back up** – open **Settings** → **[your name]** → **iCloud** → **iCloud Backup** → **Back Up Now**, or connect to a computer and run a backup through iTunes/Finder.  

### 2. Update wirelessly (directly on the iPhone)  
1. Open **Settings**.  
2. Tap **General**.  
3. Select **Software Update**.  
4. The iPhone checks for the newest iOS version available for this model.  
5. If an update is found, tap **Download and Install**.  
6. Enter your passcode when prompted.  
7. When the download finishes, tap **Install Now** (or schedule for later).  
8. The phone will restart, display the Apple logo with a progress bar, and then boot to the home screen.  

### 3. Update with a computer (iTunes on Windows/macOS Mojave or earlier, Finder on macOS Catalina or later)  
1. Connect the iPhone 6 to the computer with the Lightning‑to‑USB cable.  
2. Launch **iTunes** (or open **Finder** if on macOS Catalina or later).  
3. Select the iPhone icon (iTunes) or the device name under **Locations** (Finder).  
4. In the device summary view, click **Check for Update**.  
5. If an update is available, click **Download and Update** (or **Download** then **Update**).  
6. Enter the iPhone passcode if asked, then let the computer install the update.  
7. The iPhone will reboot automatically when the installation completes.  

### 4. Verify the update  
After the device restarts, go to **Settings** → **General** → **About**. The **Software Version** line should show the latest iOS release for the iPhone 6.

## Keeping the iPhone 6 Secure  

- **Enable a passcode** (or Touch ID) and require it after each lock.  
- **Turn on Find My iPhone** (Settings → [your name] → Find My).  
- **Use two‑factor authentication** for your Apple ID.  
- **Keep apps updated** via the App Store’s update mechanism.  
- **Limit app permissions** (Settings → Privacy) to only what’s necessary.  
- **Disable services you don’t use** (e.g., Bluetooth, Wi‑Fi, AirDrop) when not needed.  
- **Regularly back up** to iCloud or a computer so data can be restored if the device is compromised.

## End‑of‑Life Notice  

The iPhone 6 no longer receives new iOS releases beyond the final version supported for this hardware. This lack of future updates means security patches for emerging threats will not be applied, leaving the device more vulnerable over time.  

**Recommendation:** For continued protection, consider upgrading to a newer iPhone model that receives the latest iOS updates. Visit Apple’s official website to explore current models and find a replacement that suits your needs, then remove the old iPhone 6 from your home network.
