# About the Focus Devices
The focus device categories and content were generated by Gemini 2.5 Pro. Unfortunately, due to the closed source nature of many market resource firms this data was unable to be directly acquired from a reliable source, although should be sufficient for "topic selection" by the synthetic data generation pipleline. The following is a brief explanation of each CSV which will be used to guide instruction/answer pairs in the pipeline generation process.

# CSVs
## `routers.csv`
`routers.csv` contains the 20 most popular consumer routers, including routers commonly given out by ISPs.

## `smart_tvs.csv`
`smart_tvs.csv` contains the top 20 most popular smart tvs.

## `smart_hubs.csv`
`smart_hubs.csv` contains devices that are seen as a "hub" within the home network. The `smart_hubs.csv` file contains the top 20 "smart hubs" including device like Google Home Assistant, Amazon Echo, Home Pod, Nest, etc.

## `cameras.csv`
`cameras.csv` contains the 20 most popular smart security camera device such as Ring, Blink, Nest Cam, etc.

## `printers.csv`
`printers.csv` contains the 20 most popular consumer printers that are attached to the network.

## `nas.csv`
`nas.csv` contains the 20 most popular NAS storage server solutions for consumers. Due to their relative rarity, synthetic generation will not focus on this particular device list, but it would still be ideal for Ping to have insight on these devices to a certain extent, as they are by no means "rare."

## `primary_devices.csv`
`primary_devices.csv` is the star table 🌟 as it contains "primary devices" such as smartphones, PCs/Macs, and gaming consoles. It can be reasonably assumed that most consumers will have multiple devices from this table, and thus it will serve as the backbone of synthetic data generation.
