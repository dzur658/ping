# Ping
### *Make Home Networks Safer*
Repo for UC Senior Capstone

## Project Summary
Ping looks to revolutionize cybersecurity for the home of the average person in the same way Ring revolutionized the physical security of the home. Ping will combine various technologies to make security recommendations to a non-technical person in a way they can practically understand the risks of vulnerabilities and take effective action. Ping does not aim to provide 100% protection, but rather to protect users from being easy targets of mostly automated attacks.
